## 👨‍💻 My Contribution

Have you ever wondered how many tiles are actually needed before starting a construction project?  
Manual tile estimation often leads to unnecessary expenses, material shortages, and calculation errors.  

To address this real-world challenge, I developed the **Masuma Tiles Agency Calculation Project**, a smart web-based tile estimation system designed to simplify the planning process.  

The project combines modern web design with practical business requirements, enabling users to estimate tile requirements quickly, accurately, and efficiently.
